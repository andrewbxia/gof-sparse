a sparse implementation of [Conway's Game of Life](https://en.wikipedia.org/wiki/Conway's_Game_of_Life), with text input and scrolling

Input methods:
- mouse drag: place square of cells
- scroll: zoom in-out
- enter: pause simulation
- escape: delete all alive cells

a little bit of chatgpt was used in part of the input handling and some random functions of drawing pixels to the screen that are unused in the main loop anyway


preview:
- fun fact: the blue flashing effect is intentional integer overflow that's wrapping around as it subtracts every frame!
